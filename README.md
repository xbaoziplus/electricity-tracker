# 电费记账本 ⚡

一个纯本地、可离线的手机电费记录 PWA：记电表读数、记缴费、看用电趋势、按阶梯电价估算电费。

**在线使用：<https://xbaoziplus.github.io/electricity-tracker/>**

## 功能

- **总览**：本月用电量 + 环比、日均用电、当前阶梯档位与电价进度、本月估算电费、余额估算、抄表提醒
- **记录**：电表读数 / 缴费记录，增删改，读数倒走警告（换表可强制保存）
- **趋势**：近 12 个月用电量 / 费用柱状图，点柱查看月度明细
- **设置**：阶梯电价表可编辑（按年 / 按月清零）、单一电价模式、JSON 导出导入备份、示例数据

## 装到手机

1. 手机浏览器打开上面的网址
2. **iPhone**：Safari → 分享 → 「添加到主屏幕」；**Android**：Chrome → 菜单 → 「安装应用」
3. 从主屏幕图标启动：全屏、断网可用，和原生 APP 一样

## 使用要点

- **先在「设置」把电价改成你当地标准**——默认阶梯表是国网典型年阶梯（2160 / 4800 度分档），各省市不同，以你的电费单为准
- 第一次读数只是表底，记第二次后开始出用电量与估算
- 想先看效果：设置 → 「载入示例数据」，随时一键清空
- **数据只保存在本机浏览器**（localStorage），无后端、无账号；换设备前用「导出 JSON」备份
- 所有电费为估算值，以供电公司账单为准

## 技术

单文件 vanilla JS（零依赖、零构建），内联 SVG 图表与图标；Service Worker 离线缓存（应用外壳 + Google Fonts 预缓存）；Web App Manifest 支持添加到主屏幕。
